# Arduino Setup

1. Download the Arduino IDE: https://www.arduino.cc/en/software/
2. Download library zip files:
  2.1 RoboClaw Arduino: https://resources.basicmicro.com/using-the-roboclaw-arduino-library/
  2.2 ellapsedMillis: https://github.com/pfeerick/elapsedMillis
3. Follow the instructions on the [RoboClaw Arduino Library page](https://github.com/pfeerick/elapsedMillis) for adding a library. Repeat for the elapsedMillis library.
4. Add the Teensey 4.1 board to Arduino IDE: https://www.pjrc.com/teensy/tutorial.html
5. Open the `Arduino_ROS.ino` file using Arduino IDE. The other files in the project will also open in their own tab.
6. Use Arduino IDE to uplaod the code to the Teensy via a USB connection.
